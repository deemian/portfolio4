Mijn Projecten

Pushup Counter
Dit is een smart device dat ik heb gemaakt voor het eerste CLE project van de Hogeschool Rotterdam. Ik moest iets maken dat afgestemd was op mijn groepsgenoot, en omdat hij veel aan gym doet, besloot ik een apparaatje te maken dat zijn pushups kan tellen. Zet hem neer naast je schouder, en begin met pushups doen! Hij laat het aantal pushups zien door middel van drie verschillende kleuren lampjes. Rood betekent het laatste getal oftewel de eenheden. Als het rode lampje op plek 3 staat betekent dat 3. Blauw betekent de tientallen, dus als het blauwe lampje op 5 staat betekent dat 50. Groen betekent de honderdtallen, dus als het groene lampje op 1 staat betekent dat 100. Samen kunnen die drie lampjes dus getallen tot 999 laten zien. Als je toch meer dan 999 pushups kan doen, dan kan je altijd nog oneindig veel getallen toevoegen in de code. (De code is te vinden op de Github)

De Auto-Wekker
Dit is een groepsopdracht gemaakt voor CLE 2 samen met Abel, Abukar en Calvin. Het is een wekker gemaakt voor mensen die te veel snooze gebruiken. Je kan de tijd instellen met de knopjes, en met een lange klik start je de timer. Het laat het aantal minuten zien op dezelfde manier van de pushup counter. Als de tijd voorbij is gaat hij rijden, en als hij een afgrond ziet stopt hij, gaat hij terug en dan weer naar voren. De bedoeling is dat je erzelf achterna moet gaan om het te stoppen. Ik heb zelf een groot deel van de code geschreven. (De code is te vinden op de Github)
